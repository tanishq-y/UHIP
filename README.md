# UHIP – Urban Heat Intelligence Platform
ISRO Bharatiya Antariksh Hackathon 2026 – Problem Statement 1

## Quick start
1. Create env: `python -m venv .venv && source .venv/bin/activate`
2. Install: `pip install -r requirements.txt`
3. Pick city: edit `src/pipeline/config.py` (default: Delhi)
4. Run Step 1: `python src/pipeline/01_download.py`
5. Run Step 2: `python src/pipeline/02_preprocess.py`

Folder structure matches the architecture in your Executive Summary.
